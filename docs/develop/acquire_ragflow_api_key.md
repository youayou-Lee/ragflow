# 获取RAGFlow API密钥

RAGFlow服务器需要API密钥来验证您的HTTP/Python或MCP请求。本文档提供了获取RAGFlow API密钥的说明。

1. 单击RAGFlow UI右上角的头像以访问配置页面。
2. 单击**API**切换到**API**页面。
3. 获取RAGFlow API密钥：

![ragflow_api_key](https://raw.githubusercontent.com/infiniflow/ragflow-docs/main/images/ragflow_api_key.jpg)

:::tip 注意
有关RAGFlow的HTTP或Python API的完整参考，请参阅[RAGFlow HTTP API参考](../references/http_api_reference.md)或[RAGFlow Python API参考](../references/python_api_reference.md)。
:::
