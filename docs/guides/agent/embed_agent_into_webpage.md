# 将智能体嵌入网页

您可以使用 iframe 将智能体嵌入第三方网页。

1. 在继续之前,您必须 [获取 API 密钥](../models/llm_api_key_setup.md);否则,将出现错误消息。
2. 在 **智能体** 页面上,单击目标智能体以访问其编辑页面。
3. 单击画布右上角的 **管理 > 嵌入网页** 以显示 **iframe** 窗口:
4. 复制 iframe 并将其嵌入您的网页。
