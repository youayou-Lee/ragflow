# 索引器组件

一个定义如何索引块的组件。

---

**索引器**组件索引块并配置它们在文档引擎中的存储格式。

## 场景

**索引器**组件是所有摄入流水线的强制性结束组件。

## 配置

### 搜索方法

此设置配置块在文档引擎中的存储方式:全文、嵌入或两者皆有。

### 文件名嵌入权重

此设置定义文件名对最终嵌入的贡献,它是块内容和文件名的加权组合。本质上,较高的值使文件名在最终*复合*嵌入中具有更大的影响力。

- 0.1: 文件名贡献 10%(块内容 90%)
- 0.5(最大值): 文件名贡献 50%(块内容 90%)
