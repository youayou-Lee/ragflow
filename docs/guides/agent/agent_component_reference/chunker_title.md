# 标题分块器组件

一个按标题级别拆分文本的组件。

---

**标题分块器**组件是一个文本分割器，使用指定的标题级别作为分隔符来定义分块边界并创建分块。

## 场景

**标题分块器**组件是可选的，通常紧接在**解析器**之后放置。

:::caution 警告
在**令牌分块器**之后放置**标题分块器**是无效的，将导致错误。请注意，此限制目前不是系统强制执行的，需要您注意。
:::

## 配置

### 层次结构

指定用于定义分块边界的标题级别：

- H1
- H2
- H3（默认）
- H4

单击**+添加**在此处添加标题级别或更新相应的**正则表达式**字段以自定义标题模式。

### 输出

**标题分块器**组件输出的全局变量名称，可以被摄取管道中的后续组件引用。

- 默认：`chunks`
- 类型：`Array<Object>`
